# Code of Conduct

This project follows the [Contributor Covenant Code of Conduct v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Please read the full text at the link above. In summary: be respectful, constructive, and inclusive. Harassment and exclusionary behavior are not tolerated.

## Reporting

If you experience or witness unacceptable behavior, please report it by opening an issue or contacting the maintainers directly.
